# ComESP32

## En proyecto del ESP32 con WebSockets, ¿quién actua como Host (Servidor) y quién como Cliente? Define brevemente qué responsabilidad tiene cada uno en la comunicación.

el Host es la computadora del profesor y los estudiantes actuan como clientes, además el Host se presta como centro para que lleguen los mensajes de los clientes.

## explica con tus palabras qué representaría la Dirección IP y qué representaría el Puerto. ¿Por qué es necesario especificar ambos para que el programa funcione?

porque la dirección IP nos indica la ubicación exacta del servidor para que los clientes puedan ingresar, el puerto es de lo que va a tratar el servidor para permita relacionarse con los clientes de manera grata.
